# 📊 Collections Dashboard - Egada

## 🔹 Overview
This dashboard provides a comprehensive view of collections performance for Egada, focusing on customer distribution, collectors' efficiency, outstanding balances, and collection activities. It helps monitor financial performance and supports better decision-making.

---

## 🔹 Objectives
- Track number of customers per collector  
- Monitor total number of accounts  
- Analyze outstanding debt (balances)  
- Evaluate collectors’ performance  
- Provide visibility on activities and portfolio links  

---

## 🔹 Key Features

### 📌 KPIs
- Total Customers  
- Total Accounts  
- Total Outstanding Balance  
- Total Collections  

### 👥 Collector Performance
- Collection amount achieved by each collector  
- Comparison between collectors to identify top performers  

### 📊 Customer Distribution
- Number of customers assigned to each collector  

### 💰 Financial Overview
- Total outstanding balances vs collected amounts  

### 🔗 Portfolio Access
- Direct link to each collector’s portfolio for detailed follow-up  

### ⚙️ Activity Tracking
- Monitoring collectors’ activities and performance  

---

## 🔹 Filters & Interactivity
The dashboard includes dynamic filters for:
- Collector Name  
- Activity  
- Portfolio  

All visuals are interconnected for better analysis and exploration.

---

## 🔹 Tools & Technologies
- Power BI  
- Data Modeling  
- DAX  

---

## 🔹 Key Insights
- Identify top-performing collectors based on collection amounts  
- Detect collectors with high customer load  
- Highlight accounts with high outstanding balances  
- Improve collection strategies through performance tracking  

---

## 🔹 Business Value
This dashboard enables:
- Better monitoring of collection operations  
- Improved allocation of customers to collectors  
- Faster decision-making  
- Increased collection efficiency and reduced outstanding balances
